# AI Resume Analyzer

## Project Overview
This project analyzes tech resumes and predicts the job category using Machine Learning (Naive Bayes + TF-IDF).

## Dataset
- `resumes_dataset.jsonl` from Kaggle (3,500+ anonymized tech resumes)
- Dataset contains: ResumeID, Category, Name, Email, Skills, Experience, Education, Text, Source

## Technologies Used
- Python
- Pandas
- Scikit-learn
